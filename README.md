# ✅ TecBoard

Projeto desenvolvido utilizando **React**, com foco na criação de uma aplicação para gerenciamento e divulgação de eventos de tecnologia, permitindo cadastrar e visualizar eventos por diferentes categorias em uma interface moderna, responsiva e intuitiva.

## 🌐 Projeto Online

👉 **Acesse o projeto:** https://projeto-tecboard-navy.vercel.app/

## 📋 Funcionalidades

- Cadastro de novos eventos
- Exibição dinâmica dos eventos cadastrados
- Organização dos eventos por categorias
- Exibição de imagens através de URL
- Imagem padrão para URLs inválidas ou não informadas
- Formulário para cadastro de eventos
- Interface responsiva
- Componentes reutilizáveis

## 🛠️ Tecnologias utilizadas

- React
- JavaScript
- CSS3
- Vite

## 💻 Sobre o projeto

Este projeto foi desenvolvido como parte dos meus estudos em React, permitindo praticar conceitos fundamentais como componentização, gerenciamento de estado, manipulação de formulários, renderização dinâmica de listas e organização de componentes reutilizáveis.

## 🚀 Aprendizados

Durante o desenvolvimento deste projeto, pratiquei:

- Componentização com React;
- Gerenciamento de estado utilizando `useState`;
- Manipulação de formulários controlados;
- Renderização dinâmica com `map`;
- Filtragem de dados utilizando `filter`;
- Manipulação e formatação de datas;
- Tratamento de imagens com fallback utilizando `onError`;
- Estruturação de projetos React utilizando **Vite**;
- Boas práticas de organização de código;
- Desenvolvimento de interfaces responsivas com CSS.

## ▶️ Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/gabriel-pereira0/Projeto_Tecboard.git
```

2. Acesse a pasta do projeto:

```bash
cd tecboard
```

3. Instale as dependências:

```bash
npm install
```

4. Execute o projeto:

```bash
npm run dev
```

5. Abra no navegador:

```text
http://localhost:5173
```
